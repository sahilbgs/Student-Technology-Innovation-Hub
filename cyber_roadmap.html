Learning Path

Cybersecurity Roadmap

From Zero to Industry-Ready Security Professional

1

Foundations & Core Concepts

Understand how computers, networks, and operating systems work.

  • Networking Basics (OSI, TCP/IP, DNS)
  • Linux & Command Line
  • Windows Internals (AD, PowerShell)
  • Cryptography Basics
  • Programming for Security (Python)

Resources

  • TryHackMe — Pre-Security
  • OverTheWire: Bandit
  • Linux Journey
Real World Usage: The bedrock everything else builds on. Do not skip this phase.
2

Blue Team Skills (Defending)

Learn to defend systems — the skill most employers need.

  • SOC Operations (SIEM, Splunk)
  • Network Traffic Analysis (Wireshark)
  • Malware Analysis Basics
  • MITRE ATT&CK Framework
  • Vulnerability Management

Resources

  • CompTIA Security+
  • TryHackMe — SOC Level 1
  • Blue Team Labs Online
Real World Usage: Defending enterprise networks, the most common entry point.
3

Offensive Security & Ethical Hacking

Learn to think like an attacker.

  • Pentest Methodology
  • Recon & Scanning (Nmap, OSINT)
  • Exploitation Frameworks (Metasploit)
  • Web Application Hacking (OWASP)
  • Privilege Escalation

Resources

  • eJPT (eLearnSecurity)
  • HackTheBox — Starting Point
  • PortSwigger Web Academy
Real World Usage: Penetration testing, vulnerability assessments, red teaming.
4

Specialization & Advanced Skills

Pick a lane based on your interests and the job market.

  • Cloud Security (AWS/Azure/GCP)
  • Threat Hunting (KQL/SPL)
  • Malware Reverse Engineering
  • Red Teaming (Cobalt Strike)
  • Digital Forensics (DFIR)

Resources

  • OSCP (Offensive Security)
  • AWS Security Specialty
  • SC-200 (Microsoft)
Real World Usage: Mid-to-senior level specialized enterprise roles.
5

Portfolio & Job Readiness

Turn skills into proof.

  • Home Lab Setup
  • Portfolio Building (Writeups, Blogs)
  • Bug Bounty Programs (HackerOne)
  • Community Networking (DEF CON)

Projects

  • HackTheBox Rank
  • Writeups on Medium / GitHub
  • Bug Bounty Triage Reports
Real World Usage: Getting hired. A strong CTF profile often beats a resume full of certs.

Final Career Paths

  • Security Operations Center (SOC) Analyst
  • Penetration Tester
  • Cloud Security Engineer
  • Incident Responder (DFIR)
  • DevSecOps Engineer
